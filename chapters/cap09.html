Graficos¶

Graficos so uma forma muito interessante de visualizar informao, um recurso amplamente utilizado em Data Science, Engenharia, Fscia, etc... Assume-se aqui netes captulo que o aluno j possui o conhecimento prvio sobre grficos e o funcionamento dos mesmos.

Por padro o python no vem equipado com suporte criao de grficos, mas por sorte, e pelo esforo de inumeros desenvolvedores, existem vrias bibliotecas de apresentao de grficos para o python.

Neste captulo iremos utilizar a biblioteca chamada matplotlib. A matplotlib tem suporte a uma gama quantidade de grficos diferentes. Aqui nesta pagina voc pode ver os grficos que esta poderosa biblioteca possui. Como podemos ver pelo site da bibliteca, a mesma da suporte a muitos tipos de grficos, e infelismente no temos como ver todos estes grficos. Entretanto, uma coisa legal do site do matplotlib que ele possui exemplos de cdigo para cada grfico, ento muito fcil entender o funcionamento de um grfico novo.

Se voc esta utilizando um ambiente python puro ser necessrio isntalar o matplotlib, entrentanto se voc utiliza o pacoto da anaconda ele j vem incluso =).

Iremos cobrir dois tipos de grficos aqui neste captulo: Linhas e pontos.

bacana n? Agora, j podemos analizar nossa informao utilizando o grfico. Claramente um dos sexos tira notas melhores que o outro, baseado nesta informao. Mas voc concorda que fica ruim de ficar olhando o eixo y para saber qual o sexo do aluno? No seria melhor mostar cada sexo com cores diferentes?

Exemplo:

In [7]:
import matplotlib.pyplot as plt

notas = [10.0, 9.1, 3.2, 6.1, 8.9, 1.2, 0.5]
sexo  = [1,0,0,1,1,0,0]

nSexo0 = []
nSexo1 = []

sSexo0 = []
sSexo1 = []

for i  in range(0, len(sexo)):
    if sexo[i] == 0:
        nSexo0.append(notas[i])
        sSexo0.append(0)
    else:
        nSexo1.append(notas[i])
        sSexo1.append(1)

Pronto, agora temos as notas separadas por sexo, observe que mantivemos uma lista que diz qual o valor do sexo de cada pessoa, isso para manter o eixo y com esta escala.

Agora s mostrar estes dados, o matplotlib permite que sejam dados inmeros plots antes do grfico ser mostrado, portanto podemos dar um plot para o sexo0 com uma cor e outro plot com sexo1 com outra.

In [8]:
import matplotlib.pyplot as plt

notas = [10.0, 9.1, 3.2, 6.1, 8.9, 1.2, 0.5]
sexo  = [1,0,0,1,1,0,0]

nSexo0 = []
nSexo1 = []

sSexo0 = []
sSexo1 = []

for i  in range(0, len(sexo)):
    if sexo[i] == 0:
        nSexo0.append(notas[i])
        sSexo0.append(0)
    else:
        nSexo1.append(notas[i])
        sSexo1.append(1)
plt.plot(nSexo0, sSexo0, '.g')
plt.plot(nSexo1, sSexo1, '.r')
plt.show()

E agora podemos ver que cada sexo tem uma cor diferente no grfico. Neste caso podemos ver que o sexo da cor vermelha vai normalmente melhor que o da cor verde.

Mas este grfico ainda esta muito simples, podemos melhora-lo colocando um ttulo e indentificao de cada eixo.

Para adicionar um ttulo utilizamos o comando title. E para os eixos os comandos xlabel e ylabel.

Bacana n, agora vamos colocar uma legenda. Para isso s adicionarmos o comando label em cada plo, o valor de legenda destes dados e a localizao da legenda.

Para indicar a localizao da legenda utilizamos o comando plt.legend(loc='local da legenda')

Existem inmeros locais possiveis: 'upper right' : 1, 'upper left' : 2, 'lower left' : 3, 'lower right' : 4, 'right' : 5, 'center left' : 6, 'center right' : 7, 'lower center' : 8, 'upper center' : 9, 'center' : 10,

Para facilitar o calculo do coseno vamos utilizar a biblioteca de matematica, esta biblioteca da suporte a inmeras funes diferentes. Aqui voc pode ver todas as funes desta biblioteca: https://docs.python.org/2/library/math.html

Bom, vamos comear criando uma lista com os valores de cosseno. Para isso primeiro importamos o coseno da biblioteca de matematica. E depois criamos uma lista utilizando ele e um lao de repetio.

from math import cos #importamos o cosseno
from math import sin #importamos o seno

import matplotlib.pyplot as plt

fCos = [] #lista que vai guardar os valores de cosseno
fSin = [] #lista que vai guardar os valores de seno

#vamos gerar 1000 valores de coseno
for i in range(0,1000):
    valorCos = cos(i*0.01)
    valorSin = sin(i*0.01)
    
    fCos.append(valorCos)
    fSin.append(valorSin)
    
    
plt.plot(fCos, 'r')
plt.plot(fSin, 'b') #muda a cor para blue
plt.show()
